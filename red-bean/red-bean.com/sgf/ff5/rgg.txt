A new SGF standard is currently in the making.
SGF stands for 'Smart Game Format' and is a standard
for storing game records.

The new standard SGF FF[5] aims at:

- Defining property sets, in order to make it easier to
  expand current programs or create new programs.
  Continuing the work of FF[4] for a clean, consistent,
  unambiguoues SGF specification.

- Extending game-info properties to be able to store
  information not covered by FF[4], e.g. team players,
  people involved in game (referee, time keeper, ..),
  more specific information about EVent, REsult, DaTe, etc.

- supporting multiple languages in a single file

- several other extensions
