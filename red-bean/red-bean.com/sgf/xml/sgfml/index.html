SGFML - XML DTD of SGF

Here's a very early (read: too early to publish) draft of SGFML, an XML DTD based on SGF.

A rough guide for SGF properties <-> SGFML elements translation.
See CHANGES for latest developments and older versions.

Note: SGFML is just here to give you an idea and for historic purposes. It is no longer maintained. However, there's a group of people devoted to creating an XML syntax for Go games.

You can find the list at SourceForge: XGo mailing list of the XGo project

SGFML DTD:
version 0.02
 
Example files:
typical server game: SGF -- SGFML
commented game from GTL: SGF -- SGFML

Go to this XML Validator and check the XML files yourself.
Or go to IBM's DOM view of XML files.
