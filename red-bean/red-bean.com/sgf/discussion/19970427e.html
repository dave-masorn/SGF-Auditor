Date: Sun, 27 Apr 1997 20:45:43 -0400 From: fotland@ix.netcom.com Subject: RE and Date formats You have made me think about the user interface some more, and here is my plan for RE and DT formats now. For RE, I will present a set of radio buttons, similar to YAGO winner score o white xxxxx o black o resignation o unknown o time and I will also show a read-only field with the text from the file's RE property value. This field will be blank and greyed out if the file has the proper format. So the user can see the result either way. And it is easy for him to correct the format by using the radio buttons. For date I will do something similar, giving a set of MM/DD/YYYY data entry fields, and a text box for entering more information. David
