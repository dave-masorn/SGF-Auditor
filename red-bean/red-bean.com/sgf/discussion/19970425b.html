Date: Fri, 25 Apr 1997 13:11:15 -0400 From: Anders Kierulf Subject: RE: I think it's better to leave the information in the right fields even if it doesn't follow the standard formatting. At least then there's the chance for somebody to go through it manually and correct the format. If it all gets lumped into the game comment, it's much harder to clean up later. In my mind, reading in games that don't necessarily follow the correct FF[4] formatting shouldn't force you to write them back out using FF[3]. Of course, any new games should follow the standard, and if a program can automatically fix up the format, that's great. I think it may be sufficient for FF[4] to rigorously define the format for these fields, and mandate that format for new games but only strongly recommend it for old games. Anders
